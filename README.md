运行

1.以记事本方式打开set_env.bat，将current_path后面改成你的platform-tools的路径，如C:\Users\xxx\Desktop\onmoji_auto\platform-tools\
以管理员身份运行set_env.bat，按住win+R，输入cmd回车，在命令行输入"adb"测试是否成功。
若失败请自行将platform-tools加入到系统环境变量中。
2.在桌面右键-显示设置-缩放里检查是否为100%
3.启动阴阳师专版mumu模拟器，设置分辨率为超宽屏2400×1080，收起底层小工具，窗口置于上层启动阴阳师后（不要最小化）启动run。
若窗口大小自动调整且移至画面左上角即为成功。
4.启动游戏根据提示使用即可。
5.注意模拟器必须置于顶层，保证图像能正常识别。
6.后续运行从第4步开始即可。

使用

1.探索模式：进入庭院灯笼探索后开启，默认困28，否则请自行选择副本后启动。需要换狗粮请游戏内设置自动轮换。
2.队长模式：支持觉醒、御魂（大蛇）。建议手动刷一轮后，点击自动邀请队友后开启。支持超时邀请队友，请确保被邀请的队友在最近组队的第一位。
3.队员模式：支持觉醒、御魂（大蛇）。任意界面开启即可，自动接受邀请进入组队。
4.突破模式：突破界面开启。支持全刷、保级和掉级模式。
5.单刷模式：支持御魂（大蛇、业原火、日轮、永生）、觉醒、御灵。在副本入口锁定阵容后开启。
6.穿插模式：任务组包含探索、突破、休息，需要分别设定。
7.活动：一般在活动主界面运行，根据提示进行设定。

常见问题

1.运行onmoji_auto.exe后闪退
请确保已经运行mumu最新版并启动游戏，置于前台

2.一直初始化失败，且雷电模拟器窗口无变动
（1）设置雷电模拟器分辨率为2400×1080
（2）将雷电模拟器置于最上层
（3）桌面右键-显示设置-缩放与布局，更改为100%后重启模拟器
注：若不想设置为100%，可联系作者进行修改。（但需要自己截取适配图）

3.模拟器连接出现问题
（1）命令行输入adb devices查看设备是否存在
（2）模拟器内设置ADB调试为“开启本地链接”

4.是否可以隐藏到后台
为降低截图延迟，工具使用win32的截图来识别，实测截图延迟（1.8s→0.2s），以防止识别内容座位和实际位置的偏差。
因此不能直接最小化。建议用win+tab进行后台挂机
